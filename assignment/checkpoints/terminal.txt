# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?
Running the command 'ls' in an empty folder would return with zero results  because there is nothing to read inside of the folder.
If you run the command 'ls' on a folder containing files, the terminal would show you all the files that are held in the folder.
